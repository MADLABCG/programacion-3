Elementos HTML

  1. Anchor Tags <a href="https://mi.enlace.com" tagert="_blank">Nombre que se vera en el Navegador</a>
    • href la direccion url del hipervínculo donde queremos redireccionar
    • target esta propiedad define el comportamiento de donde se abrirá el hipervínculo seleccionado
      • Yo te llevo a Facebook
  2. Imagen <img src="archivo_de_imagen" alt="texto a mostrar en caso de no estar disponible el archivo de imagen">
    • src la ruta donde se encuentra la imagen, puede ser una url o la ubicacion de nuestro archivo de imagen dentro del proyecto
    • alt Texto alternativo si la visualización de la imagen falla por alguna razon
  3. Negritas <b>Aqui va el texto que queremos en negritas</b>
  4. Brake Line (Nueva linea, se usa al final de un texto o elemento al que queremos dar una nueva linea debajo de este) <br>
  5. Listas
    • Lista Ordenada <ol></ol>
      • Dentro de una lista ordenada tenemos los elementos de la lista <li>Elemento que queremos mostrar</li>
    • Lista Desordenada <ul></ul>
      • Dentro de una lista desordenada tenemos los elementos de la lista <li>Elemento que queremos mostrar</li>
    • Ejemplo
      1. Primer Punto
        1. Sub Punto
        2. Sub Punto
      2. Segundo Punto
        • Sub Punto
        • Sub Punto
      • Primer Punto
        1. Sub Punto
        2. Sub Punto
      • Segundo Punto
